# AGLI Tutorial

## Goal

Use AGLI to turn verified social/video performance examples into safer draft content for Amsterdam Game Lab and Pro Actief.

## 1. Collect Content Examples

Use low-cost discovery sources:

- LinkedIn analytics and manual competitor review
- YouTube search, YouTube Studio, VidIQ, or TubeBuddy
- Instagram insights or manual post review
- TikTok Creative Center or manual public metric review

For each item, capture:

- Platform
- Title or hook
- Creator
- URL
- Topic
- Angle
- Hook type
- Format
- Visual style
- Length in seconds
- Audience
- Views
- Engagements
- Comments
- Shares
- Metric date

## 2. Import Data

Open AGLI and click **Import CSV**. Use `public/sample-import.csv` as the template.
The app starts empty, so the ranking and pattern engine only use rows you import.

## 3. Read the Ranking

AGLI scores each item using:

- View signal
- Engagement rate
- Share signal
- Comment signal
- Recency
- Source quality

Click a strong row to make it the active reference for Draft Studio.

## 4. Review Patterns

Look at:

- Winning hooks
- Recurring topics
- Formats and lengths
- Visual styles

The point is not to copy a post. The point is to understand the repeatable structure behind traction.

## 5. Generate Drafts

In Draft Studio:

- Choose the platform.
- Pick the pattern.
- Adjust the Pro Actief product angle.
- Copy the draft.

Before publishing, review:

- Accuracy
- AGL tone of voice
- Legal and brand risk
- Whether any client proof line is verified
- Whether the content still feels credible, not clickbait

## 6. Export

Click **Export analysis** to download a JSON file with ranked rows, pattern summary, generated draft, and guardrails.
